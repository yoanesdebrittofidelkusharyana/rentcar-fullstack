# RentCar Web Application

Full-stack car rental web application built as part of an Associate Member assessment.

## Tech Stack
- Frontend: React (Vite)
- Backend: ASP.NET Core C# (.NET 8)
- Database: SQL Server

## Features
- User authentication (Login & Register)
- Car availability search
- Rental transaction
- Transaction history

## Notes
This project is created for learning and assessment purposes.
